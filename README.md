# Manipulação de Arquivos em Python - Projeto Completo

## Descrição

Este projeto reúne exercícios práticos de **leitura, escrita e manipulação de arquivos em Python**.
Cada exercício explora diferentes aplicações do uso de arquivos `.txt` e `.csv`, consolidando o aprendizado sobre entrada, saída e processamento de dados.

## Funcionalidades

1. **Criação e leitura de perfil (`perfil.txt`)**
2. **Diário pessoal** – registra sentimentos do usuário
3. **Lista de tarefas** – adiciona e lista tarefas salvas
4. **Contador de linhas** – conta quantas linhas há em um arquivo
5. **Copiador de arquivos** – duplica o conteúdo de um arquivo para outro
6. **Tabuada** – gera um arquivo com a tabuada de um número
7. **Calculadora de média** – lê notas de um arquivo e calcula a média
8. **Buscador de palavras** – procura quantas vezes uma palavra aparece em um arquivo
9. **Agenda de contatos (`agenda.csv`)** – adiciona, lista e busca contatos por nome

## Tecnologias Utilizadas

* Python 3
* Manipulação de arquivos (`open`, `read`, `write`, `append`)
* Estruturas condicionais e laços de repetição
* Conversão de tipos e listas

## Objetivo do Projeto

Praticar o uso de **arquivos de texto e CSV**, além de consolidar conceitos de entrada de dados, loops e condicionais em Python de forma interativa e aplicada.
